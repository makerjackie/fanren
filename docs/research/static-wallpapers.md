## 排除和谨慎项

- B站上大量“4K / 6K / 8K 超清壁纸”视频来自个人 UP 主，部分页面标注“未经作者授权，禁止转载”，且下载链路可能在充电动态或外部网盘。此类资源不适合作为主站官方资源目录入口。
- Reddit、Pinterest、普通壁纸站、搬运图站可作为线索，但不建议收录为高质量候选，除非能追溯到官方发布页。
- 新闻站转载的图片页只适合做公开信息链接，不能替代版权授权。

## 推荐落地方式

1. 首页或资源页先使用“官方链接卡片”模式：标题、来源、发布日期、资源类型、跳转按钮。
2. “官方壁纸”分区优先放 B站官方动态第 3、4 条。
3. “场景设定”分区优先放微博官方第 8、9、10 条。
4. “番剧 / 剧集入口”分区放 BiliBili、B站、优酷正版页。
5. 若后续需要站内图片墙，先整理授权申请清单，不要从上述页面直接复制图片进 `public`。
